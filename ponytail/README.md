# Ponytail Version

This version uses the Ponytail mindset:

> Use the Ponytail mindset: write the least code necessary, avoid speculative abstractions, avoid new dependencies unless absolutely required, and prefer boring standard-library solutions.

## Run

Open `index.html` directly, or run:

```bash
python3 -m http.server 8001
```

Then open `http://localhost:8001`.

## Notes

- Single HTML file.
- No dependencies.
- No build step.
- Uses `localStorage`.
- Only implements add, list, done, and delete.
